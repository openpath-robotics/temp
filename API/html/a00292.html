pho::api::CameraMatrix64f pho::api::PhoXiCalibrationSettings::CameraMatrix

Camera's 3x3 matrix describing the mapping from 3D (world) to 2D (image) corresponding to the native resolution. Note: The scanning mode appropriate camera matrix is sent as part of the FrameInfo structure. Note: Prior PhoXi Control 1.8.1, the camera matrix in FrameInfo structure corresponded to the native resolution.
