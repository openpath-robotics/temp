◆ NormalsEstimationRadius

int pho::api::PhoXiProcessingSettings::NormalsEstimationRadius

Estimates the radius of normal estimation. Higher values (4 max -> 9x9 neighborhood) offers smoother normals, while smaller values (1, 2) is good for retrieving smaller depth details. Affects normal map generation.
