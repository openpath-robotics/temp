bool pho::api::PhoXiFeatureInterface::isEnabled ( ) const

Tells if specific feature is enabled for this camera.

Some features can be implemented for a specific Device class, but are not enabled because of a minor difference between the instances of the same family.

Returns
true if the feature is enabled for the specific device, returns false otherwise.

◆ isImplemented()

bool pho::api::PhoXiFeatureInterface::isImplemented ( ) const

Tells if specific feature is implemented for this camera.

Some features can be Hardware dependent and could be present only for specific families of products. If the feature is not implemented for a Device Type, no instance of the family will have the feature.

Returns
true if the feature is implemented for the specific device type, returns false otherwise.
