Detailed Description

Timeout format in ms.

PhoXiTimeout is used in all calls when Timeout have sense calls Like GetFrame takes Timeout as optional parameters to wait for the frame. If default param is supplied, timeout supplied as PhoXiDevice->Timeout will be used.
