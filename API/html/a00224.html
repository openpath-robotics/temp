Detailed Description

Defines the used coordinate space.

Currently, only Freerun and Software Trigger are supported. The type is also used as a smart enumerator. PhoXiDevice->CoordinateSettings->CoordinateSpace = PhoXiCoordinateSpace::CameraSpace.
