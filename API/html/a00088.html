Detailed Description

template<class DataType, int ChannelCount>
class pho::api::MatElementInterface< DataType, ChannelCount >

Interface of a Matrix element object with specified Channels count.

The matrix element is used to carry multi, or single channel types, like XYZ for the PointCloud. The interface does not carry any data, just the information about the type.
