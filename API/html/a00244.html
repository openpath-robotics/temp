Detailed Description

Defines the used texture source.

Select a source of the texture. Computed is a texture computed from the structured patterns - no additional image is necessary -> Lower Quality. LED will trigger an additional image with LED flash that offers an speckle less 2D image. Laser will trigger an additional image with Laser flash. Focus will set a structured pattern as a texture. Useful for analyzing problems with signal contrast and the setup of the shutter parameters. Color option is only available for Scanners with an additional color camera. Default is LED.
